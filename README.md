# World-in-Hand Navigation for Three.js
Developed as part of the course "3D Computer Graphics: Extending the Three.js Framework" at Hasso-Plattner-Institut.

## Tech Stack

- Three.js
- TypeScript
- Vite

## CLI Commands

Installation

```bash
npm i
```

Run dev mode

```bash
npm run dev
```

Build

```bash
npm run build
```

Run build

```bash
npm run preview
```
